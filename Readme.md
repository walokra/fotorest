Imgur browser app with different Java EE Frameworks
============================================
Fotorest is a simple Java EE Framework test app done with different frameworks.

* Vaadin 7
* JSF 2 + PrimeFaces
* Wicket 6

fotorest-vaadin
---------------
* Vaadin 7 (http://vaadin.com/)

fotorest-jsf2-pf
----------------
* JSF 2.0
* PrimeFaces 3.5

fotorest-wicket
---------------
* Wicket 6 (http://wicket.apache.org/)
* wicket-jquery-ui
